# 📋 ReceiptSnap Pro 隐私政策部署指南

## 🚀 手动部署步骤

### 步骤1: 创建 GitHub 仓库
1. 访问 https://github.com/new
2. 仓库名称: `ReceiptSnapPro-Privacy`
3. 描述: "ReceiptSnap Pro Privacy Policy"
4. 选择: Public (公开)
5. ☑️ 勾选 "Add a README file"
6. 点击 "Create repository"

### 步骤2: 上传文件
1. 在仓库页面点击 "Add file" → "Upload files"
2. 上传以下文件:
   - `index.html` (隐私政策页面)
   - `_config.yml` (GitHub Pages配置)
   - `README.md` (说明文档)
3. 提交更改

### 步骤3: 启用 GitHub Pages
1. 进入仓库 "Settings" → "Pages"
2. 在 "Build and deployment" 部分:
   - Source: Deploy from a branch
   - Branch: main (/root)
3. 点击 "Save"
4. 等待几分钟，页面会自动部署

### 步骤4: 获取访问地址
部署完成后，您将获得一个类似这样的地址:
`https://chengpeng111.github.io/ReceiptSnapPro-Privacy/`

## 🔗 在 App Store Connect 中使用

### 隐私政策链接
在 App Store Connect 中填写隐私政策链接时，使用:
```
https://chengpeng111.github.io/ReceiptSnapPro-Privacy/
```

### 测试链接
部署后，请测试链接是否可以正常访问。

## 🔄 更新流程

### 更新隐私政策
1. 修改 `index.html` 文件
2. 提交到 GitHub 仓库
3. GitHub Pages 会自动重新部署

### 版本控制
建议每次重大更新时:
1. 在页面顶部更新版本号和日期
2. 在 README.md 中添加更新记录
3. 保留历史版本备份
